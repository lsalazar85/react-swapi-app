# React Swapi App

React App with Swapi Api

## Run
```
    yarn install
    yarn start
```

```
    npm install
    npm start
```
Starts on `http://localhost:3000`

## Build
```
    npm run build
```

## Package.json dependencies
* React.js
* React-redux
* React-detect-offline
* Redux
* Redux-thunk
* Prop-Types
* Sass
* Material UI
